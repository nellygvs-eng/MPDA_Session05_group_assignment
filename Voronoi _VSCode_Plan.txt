## Plan para generar el código de Surface y Voronoi en Python/Grasshopper

TL;DR: crear un script Python para Grasshopper que genere una superficie rectangular, distribuya puntos, calcule Voronoi, obtenga centroides, seleccione celdas cercanas a una curva, remapee valores, escale, haga offsets, genere superficies y extruya.

### Pasos principales

1. Definir entradas
   - ancho (float)
   - profundidad (float)
   - número de puntos N (int)
   - curva de atracción (Geometry/Curve)
   - número de puntos más cercanos M (int)
   - dominio remap start/end (float)
   - offset distance (float)
   - altura (float)

2. Crear la superficie rectangular de trabajo

3. Generar N puntos dentro del rectángulo

4. Calcular el diagrama de Voronoi 2D

5. Calcular centroides de cada celda

6. Encontrar los M centros más cercanos a la curva

7. Crear cull pattern y filtrar la lista de Voronoi

8. Remapear las distancias a la curva

9. Escalar las geometrías seleccionadas desde su centro

10. Generar offset de cada geometría Voronoi escalada

11. Crear superficies con Boundary Surface

12. Calcular líneas SDL usando distancia * altura

13. Extruir las superficies resultantes

### Archivos sugeridos

- testnelly.py — posible punto de partida
- `grasshopper_voronoi.py` — script Python nuevo recomendado

### Verificación

- Comprobar que la superficie y puntos se crean correctamente
- Confirmar Voronoi y centroides
- Validar selección de M puntos cercanos a la curva
- Verificar remapeo, escala, offset, superficies y extrusiones
